IMBus


Features
  • Connections for all inductive probe types Digital technology with linearisation possibility for inductive probes
  • Connections for all incremental probes, linear scales and rotary encoders
  • Connections for all pneumatic gauge heads
  • Connections for all analogue signals ( current, voltage, ... )
  • Connections for all serial interfaces ( universal programmable Transfer and Data format )
  • Connections for all parallel interfaces ( BCD, Binary, ... )
  • Galvanic separated outputs and inputs ( PLC compatible )
  • Flexible PC and PLC connection by USB, RS232, Ethernet, Wireless LAN, Profibus
